Register

We ask for your RIT username and password to verify that you're an IGM student and retrieve basic directory information.

You'll use a username and password that you create to log into the game. Your username will not be displayed to other users, and it cannot be changed after you create it.

Optional: You can add a display name (or nickname) for other players to see, and a profile photo. These can be changed at any time.

Profile Picture

*You have the option to display a profile picture. You have the ability to change your picture at any time.
